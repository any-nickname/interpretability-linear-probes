# Public Export Notes

This export is intentionally incomplete.

Included:

- project overview and methodology notes;
- selected pipeline scripts;
- aggregate metrics, reports, and figures.

Excluded:

- raw and processed prompt datasets;
- generated model responses;
- activation tensors and trained probe weights;
- logs, local settings, temporary files, and workspace history.

Reason: the omitted files are either large, local-only, unreproducible without
model/data access, or contain raw safety-evaluation prompts/responses that
should not be published casually.
